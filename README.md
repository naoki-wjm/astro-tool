# astro-tool

占星術計算ツール。LLMに渡して解釈させる用途を想定。

## 本番URL

https://astro-tool-nine.vercel.app/

## 機能

### ネイタルチャート
- 10天体（太陽〜冥王星）+ ノースノード
- ASC / MC / ハウス（プラシーダス）
- メジャーアスペクト5種（☌⚹□△☍）

### 年間概要
- 逆行期間
- 星座イングレス（外惑星）
- ネイタルへのトランジット

### トランジット（特定日）
- 指定日の天体位置
- ネイタルへのアスペクト

### ルナリターン
- 指定年月のルナリターンチャート
- 現在地で計算
- 月2回ある場合は両方出力

### ソーラーリターン
- チャート表示
- 1年間の動き（ソーラーリターン起点の年間概要）

### シナストリー
- 2人のネイタル比較
- 相互アスペクト（天体同士、天体×ASC/MC）

## オーブ設定

- ネイタル / リターン系: 5°
- トランジット / 年間概要: 1°
- シナストリー: 3°
